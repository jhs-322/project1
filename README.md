# project1
사이트의 크립토재킹 코드를 clova api로 탐지

```git clone https://github.com/jhs-322/project1.git```\n
```cd project1```\n
```pip install -r requirements.txt```\n
```python Main.py```\n
1. [INPUT] 검사하고 싶은 사이트의 url
   [OUTPUT] JS 추출 텍스트 파일
2. [INPUT] 추출된 JS코드 텍스트 파일을 입력
   [OUTPUT] 크립토재킹 코드를 검사
3. [INPUT] 검사 내용이 담긴 텍스트 파일을 입력
   [OUTPUT] 최종 위험도 판단
